# Aether Telegram Bot

Telegram bot untuk screening coin menggunakan GMGN Skills.

## Setup
1. Clone repo
2. `npm install`
3. Buat `.env` dari `.env.example`
4. Install gmgn-cli globally: `npm install -g gmgn-cli`
5. Deploy ke server/VPS.

## Fitur Utama
- `/screen <CA>` ŌåÆ Screening lengkap token
- `/filter` ŌåÆ Atur filter coin
- Cron job otomatis screening trending coins
